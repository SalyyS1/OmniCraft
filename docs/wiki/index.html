Core

What OmniCraft does

OmniCraft creates MMOItems-style crafting menus with category browsing, recipe previews, requirements, craft delays, and server-side transactions.

Setup

Installation

  1. Build the jar with Gradle.
  2. Place the matching jar in the server plugins folder.
  3. Start the server once, then edit config files.
.\gradlew.bat clean build

Safety

Anti-dupe model

OmniCraft cancels GUI clicks, ignores risky client actions, locks player recipe transactions, checks inventory server-side, and rolls back if output cannot be delivered.

Admin

Admin tools

Use settings, browse, reload, and debug commands to inspect recipes, reload YAML, and dry-run requirements before players craft.

Help

Troubleshooting

Run /oc reload after editing files. If MMOItems or Vault is missing, recipes using those hooks should be disabled or adjusted.

Hooks

Integrations

MMOItems is used for MMO item output and matching. AdvancedEnchantments is used to apply custom enchants. Vault handles money. PlaceholderAPI handles custom conditions.

Safety

Anti-dupe model

OmniCraft cancels GUI clicks, ignores risky client actions, locks player recipe transactions, scans inventory server-side, and rolls back when output cannot fit.

Help

Troubleshooting

Use /oc validate after editing recipes. Use /oc reload after fixing files. Read logs/craft-history.log for success, fail, and rollback details.

# Wiki OmniCraft

Dùng web wiki tại `docs/index.html` hoặc GitHub Pages để xem hướng dẫn đầy đủ.

## Liên kết nhanh

- Cài đặt
- Lệnh và quyền
- Hướng dẫn craft cho người chơi
- Hướng dẫn GUI admin
- Hướng dẫn config
- Ví dụ YAML recipe
- Tích hợp MMOItems và AdvancedEnchantments
- Ghi chú chống dupe và an toàn
- Xử lý lỗi và FAQ
